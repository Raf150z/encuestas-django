# Sistema de Encuestas con Django

Aplicación web de encuestas y votaciones desarrollada con Django.

## Características

- ✅ Creación y gestión de encuestas desde panel admin
- ✅ Múltiples opciones por pregunta
- ✅ Sistema de votación con validación
- ✅ Resultados con gráficos (barras y pastel)
- ✅ Prevención de votos duplicados por sesión
- ✅ Exportación a CSV
- ✅ API REST
- ✅ Diseño responsive
